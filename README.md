# 🔱 尖刀排小组积分管理系统

基于班级小组积分考核规则搭建的一体化管理系统，实现积分录入、自动统计、可视化展示、AI智能分析。

## 📁 项目结构

```
小组积分系统/
├── main.py          # 主应用程序
├── install.bat      # 依赖安装脚本
├── run.bat          # 启动脚本
├── 积分数据.json     # 积分数据存储文件
├── 历史周数据/       # 历史周数据目录（自动创建）
└── README.md        # 项目说明文档
```

## 🚀 快速开始

### 1. 安装依赖

双击运行 `install.bat` 或在命令行执行：
```bash
pip install streamlit pandas numpy matplotlib seaborn scikit-learn openpyxl
```

### 2. 启动系统

双击运行 `run.bat` 或在命令行执行：
```bash
streamlit run main.py
```

系统会自动打开网页浏览器，显示积分管理界面。

## 📋 功能模块

### 1. 数据录入模块
- 支持录入5项积分：出勤积分、单词打卡、群分享、期末学业、单科状元
- 支持数据编辑、搜索筛选
- 支持一键导出Excel格式

### 2. 自动统计模块
- 自动计算总积分
- 自动排名排序
- 显示小组总数、平均分、最高分等统计信息

### 3. 数据可视化模块
- 排名柱状图：查看各组积分高低
- 饼图：各积分项得分占比
- 热力图：分析不同积分项之间的关联关系

### 4. AI智能分析模块
- K-Means聚类：自动划分优秀/中等/待提升梯队
- 强弱项识别：定位每组优势和薄弱项
- 智能建议：针对不同梯队给出提升方向

## 📊 积分规则

1. **出勤积分**：每周日15:00按时到校打卡，个人单次打卡计5分；小组全员准时到校，额外奖励小组30分。

2. **单词打卡**：按小组扇贝单词累计打卡总量排名，前三名依次奖励12分、8分、5分。

3. **群分享**：每人每次积1分。

4. **期末学业**：
   - 单科成绩：小组单科均分第一，奖励25分
   - 总成绩排名：1-6名依次对应80、65、55、40、20、10分

5. **单科状元**：获评年级单科状元，个人积20分，小组积10分。

## 🛠 技术栈

- **语言**：Python 3.8+
- **框架**：Streamlit（网页交互看板）
- **数据处理**：pandas、numpy
- **可视化**：matplotlib、seaborn
- **AI算法**：scikit-learn（K-Means聚类）

## 🔧 运行环境要求

- Python 3.8 或更高版本
- 支持的操作系统：Windows、MacOS、Linux
- 浏览器：Chrome、Firefox、Safari、Edge

## 📝 使用说明

1. 启动系统后，在"数据录入"标签页添加或编辑小组积分数据
2. 在"统计排名"标签页查看自动计算的积分排名
3. 在"可视化"标签页查看各类图表
4. 在"AI分析"标签页查看智能分析结果和提升建议
5. 点击"导出Excel格式"按钮可下载数据文件

## 📌 注意事项

- 数据会自动保存到 `积分数据.json` 文件
- 首次运行需要联网安装依赖包
- 建议定期导出数据备份
